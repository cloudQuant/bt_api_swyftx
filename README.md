# bt_api_swyftx

Swyftx exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_swyftx
```

## Usage

```python
from bt_api_swyftx import SwyftxExchangeDataSpot, SwyftxRequestDataSpot
```

## Features

- REST API support
- Bearer token authentication
- Symbol format: `BTC-AUD`
